# Response Note

- このフォルダ内のコードやドキュメントを編集する際は、必ず `CODE_REVISION.md` とこのメモを確認し、共通の方針を守ってください。
- 変更箇所の近くに `CodeRevision: <ID> (<説明>) (YYYY-MM-DD HH:MM)` を残すことで、バージョンと日時の両方を明示します。
- Claude Code などの外部エージェントに引き継ぐときも、ここで決めた記録形式を伝えてください。

## CodeRevision 運用の手順

1. 追加改修では新しい `CodeRevision` を作り、該当ファイルとその周辺に同じコメントを残してください（例: `CodeRevision: INC-2025-00002-R1 (2025-11-10 17:32)`）。
2. ここ `response_note.md` と `CODE_REVISION.md` の両方を更新し、対象ファイル・目的・日時をまとめて記録してください。
3. 今後のセッションでもこのフォーマットを継続し、どのタイミングでどんな変更があったかが追跡できるようにしてください。
