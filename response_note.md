## バージョントラッキング
- Rogue フォルダ内のコード修正では、変更箇所の近くに `CodeRevision: <ID> (<説明>)` を残し、バージョンを明示してください（例: `CodeRevision: INC-2025-00002-R1`）。
- このファイルおよび `CODE_REVISION.md` に最新の `CodeRevision` の履歴を記録し、他のセッションでも方針が見られるようにします。
- Claude Code や他のエージェントがこのフォルダで作業するときも、このファイルを参照すれば同じ運用を理解しているものとします。
